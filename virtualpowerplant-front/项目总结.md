# 邢台市虚拟电厂数据可视化平台项目总结

## 📋 项目概览
![描述文本](./img/home.png)
这是一个专为邢台市定制的虚拟电厂数据可视化平台，基于React技术栈开发的现代化能源管理系统。系统集成了实时监控、数据分析、设备管理、交易中心和用户用电模型等核心功能，为邢台市虚拟电厂建设提供全面的数字化支撑。

## 🏗️ 技术架构

### 前端技术栈
- **框架**: React 18.2.0 + Vite 4.0.0
- **路由**: React Router DOM 6.5.0
- **可视化**: ECharts 5.6.0 (图表库)
- **特效**: React TSParticles 2.6.0 (粒子动画)
- **样式**: CSS3 + 渐变效果 + 动画

### 项目结构
```
virtualpowerplant-front/
├── src/
│   ├── App.jsx                    # 主应用组件
```

## 🎯 核心功能模块详解

### 1. 🔐 用户认证系统 (Login)

#### 功能特性
- **多角色权限管理**: 支持4种用户角色
  - `admin` (管理员): 完整系统权限，跳转到主控台
  - `operator` (操作员): 运营管理权限，跳转到能源管理系统
  - `viewer` (查看者): 只读权限，跳转到案例研究
  - `user` (普通用户): 基础权限，跳转到主控台

#### 安全机制
- **表单验证**: 用户名最少3字符，密码最少6字符
- **实时校验**: 输入时实时清除错误提示
- **登录状态管理**: localStorage存储用户信息和token
- **路由保护**: ProtectedRoute组件保护敏感页面

#### 用户体验
- **粒子背景**: TSParticles科技感粒子动画
- **动画效果**: 页面加载动画、表单渐入效果
- **响应式设计**: 适配不同屏幕尺寸
- **错误提示**: 友好的错误信息显示

#### 预设账户
